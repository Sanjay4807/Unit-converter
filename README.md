# Python Unit Converter

A menu-driven Python application that converts fundamental physics units across multiple categories.

## Features
- Length conversion (mm, cm, m, km)
- Mass conversion (mg, g, kg, tonne)
- Time conversion (seconds, minutes, hours)
- Temperature conversion (Celsius, Fahrenheit, Kelvin)
- Energy conversion (Joule, Calorie, Watt-hour)
- Pressure conversion (Bar, Pascal, Torr)

## Technologies Used
- Python 3
